# Advent of Code 2024

_When they zag, you zig._

## Prerequisites

Add a `.env` file with an adventofcode session cookie

```bash
SESSION= # add session token here
```

## Building

```bash
make
```

## Running

```bash
cd ./build/bin/
./aoc-2024 1 # or any other day ∈ [1,25]
# or
make run ARGS=1 # or any other day ∈ [1,25]
```
