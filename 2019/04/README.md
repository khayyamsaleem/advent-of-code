## AOC 2019 Day 4

## Prereqs
Create `.env` file with:

```
session=<aoc session cookie from browser>
```


## Run

Create a virtualenv with:

```
python3 -m venv venv
```

Activate it:

```
source venv/bin/activate
```

Install deps:

```
pip install -r requirements.txt
```

Run:

```
python 04.py
```

## Run with Docker

```bash
docker build . -t ham-aocd4
docker run ham-aocd4
```
