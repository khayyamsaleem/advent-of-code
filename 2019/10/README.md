## AOC 2019 Day 10

## Prereqs

### Environment
Create `.env` file with:

```
session=<aoc session cookie from browser>
```

### Dependencies

```bash
brew install rustup
rustup-init
```

## Run

```bash
cd aoc_2019_day_10
cargo run
```
