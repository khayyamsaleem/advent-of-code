## AOC 2019 Day 8

## Prereqs

### Environment
Create `.env` file with:

```
session=<aoc session cookie from browser>
```

### Dependencies (without Docker)

```bash
brew install stack
stack setup
```

## Run

```bash
stack run
```

## Run with Docker

```bash
docker build . -t ham-aocd8
docker run ham-aocd8
```
